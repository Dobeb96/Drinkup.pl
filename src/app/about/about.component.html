Drink Up!
Search for
drink recipes

Drinkup.pl is a special site where you can find recipies and build your own drink. Our application allows you to choose ingredient and the type of drink so you can create cocktails, shots or even non alcoholic drinks. Drink Up was created for a final project from subject Programming Single Page Applications.

Technology Stack

  • Angular 5
  • AJAX
  • API TheCocktailDB
